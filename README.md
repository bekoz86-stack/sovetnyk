# Альфа-Советник (демо-прототип)

Веб-прототип бизнес-советника для холдинга: финансы, контрагенты, договоры, идеи, клуб клиентов.

## Локальный запуск

```bash
npm install
npm run dev
```

Откройте http://127.0.0.1:5173/

Автотур 30 с: http://127.0.0.1:5173/?demo=1

## Публикация в интернет (Vercel)

**[Пошаговая инструкция для всей команды → docs/VERCEL_DEPLOY.md](docs/VERCEL_DEPLOY.md)**
